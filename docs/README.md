# Sistema de gerenciamento de tarefas — TechFlow Solutions

## Objetivo
Construir um sistema ágil de gerenciamento de tarefas para uma startup de logística, permitindo acompanhar o fluxo de trabalho em tempo real, priorizar tarefas críticas e monitorar o desempenho da equipe. O projeto aplica conceitos de Engenharia de Software com foco em metodologias ágeis, versionamento, testes automatizados e integração contínua.

## Escopo inicial
- Funcionalidades essenciais:
- CRUD de tarefas: criar, listar, atualizar e remover.
- Login básico: autenticação simples de usuários.
- Validação: impedir criação de tarefas sem título.
- Mudança de escopo simulada: adicionar prioridade às tarefas (campo priority).
